# Create a window layout using FancyZones

1. Open **PowerToys**.

2. Select **FancyZones**.

    ![Screenshot](create-window-layout01.png)

3. Click **Open layout editor**
   *(Shortcut: Win + Shift + ` )*.

    ![Screenshot](create-window-layout02.png)

4. Click **Create new layout**.

    ![Screenshot](create-window-layout03.png)

5. Enter a **Name**.

6. Select the layout type:

    * **Grid** – stacked rows or columns
    * **Canvas** – freely positioned zones

7. Click **Create**.

    ![Screenshot](create-window-layout04.png)
