# Orbit around the selected object

Rotate the camera around the selected object.

## Steps

1. Select an object in the **Viewport** or **Outliner**.
2. Press **F** to focus on the object.
3. Hold **Alt**.
4. Hold the left mouse button and move the mouse to orbit around the object.

## Optional

* Hold **Alt** + middle mouse button to pan.
* Hold **Alt** + right mouse button to zoom.
