# Packages
		
| Package | Description |
|----------|----------|
|  mkdocs  | The core engine — turns Markdown into a website.  |
| mkdocs-material  | Theme: beautiful typography, dark mode, search, icons, diagrams (including Mermaid!), callouts, tabs, grids — the works.  |
| mkdocs-awesome-pages-plugin  | Lets you organize pages in folders without manually declaring everything in mkdocs.yml.  |
| mkdocs-glightbox  | Adds a sleek lightbox for images — perfect for documentation with screenshots or diagrams.  |
