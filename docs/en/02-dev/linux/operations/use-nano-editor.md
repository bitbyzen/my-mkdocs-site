# Use the nano editor

1. Open the Ubuntu terminal.
   ![Screenshot](../00-images/use-nanoeditor01.png)

2. Create a new file and open it:

      ```bash
      nano Dockerfile
      ```
   ![Screenshot](../00-images/use-nanoeditor02.png)

4. Write your content, then press **Ctrl + O** to save the file.
   ![Screenshot](../00-images/use-nanoeditor03.png)

5. Press **Enter** to confirm the file name.
   ![Screenshot](../00-images/use-nanoeditor04.png)

6. Press **Ctrl + X** to exit.
   ![Screenshot](../00-images/use-nanoeditor05.png)
