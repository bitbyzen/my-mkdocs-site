# 個人ユーザーとして登録する方法
## まとめ
サインイン画面で**Personal**を選択する。  
（※デフォルトでは**Work**が選択されているので要注意。）
![screenshot](images/signin01.png)

## 補足
Docker Desktopを使用したいだけなら、登録しなくても大丈夫。  
ただし、Docker Desktopを起動するたびに、「ログインしていません」みたいなメッセージが表示されるので、少しうっとうしい。  
このメッセージをオフにする方法は、調べた限り、**ない**。  
（設定ファイルを手動で改変すれば、消えるらしいけど、そこまではしたくはない。）  
仕方ないので、とりあえず、登録してログインすることにした。
