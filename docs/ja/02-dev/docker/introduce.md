# Dockerを導入する方法
## はじめに
ざっくり言うと、以下のとおり。

## 手順
1. WindowsにLinuxをインストールするために、WSL2をインストールする。

1. Linux（例えば、Ubuntu）をインストールする。

1. Docker Desktopをインストールする。
    - Docker Desktopは、Dockerを管理するためにローカルにインストールするアプリ。

1. Docker Desktopを起動する。
    - Docker Desktopを起動すると、ウィンドウ画面の右上に**Sign in**ボタンが表示されるけど、サインインする必要はない。  
    単にDocker Desktopを起動するだけでOK。

1. 起動したら、邪魔なので、Docker Desktopを閉じる。
    - 閉じるボタンをクリックすれば、OK。  
    閉じるボタンをクリックしても、Docker Desktopはバックグラウンドで動き続ける。
