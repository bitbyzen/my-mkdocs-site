# Ubuntu Terminalを起動する方法

## まとめ
主に以下の3つの方法がある。

| 方法 | 内容 |
| -- | -- |
| VS Codeから起動する | VS Codeを利用する場合、これがベスト。 |
| PowerSellから起動する | VS Codeを利用しない場合、これがベスト。 |
| アプリリストから起動する | これだとシステム依存になるので、開発では基本的に使用しない。</br> （※新しいOSなら、たぶん問題ないけど。）|
